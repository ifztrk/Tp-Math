# TP Python : Recherche, Tri et Graphiques

Dans ce TP, j’ai appris à :

faire des fonctions qui cherchent un nombre dans une liste  
faire des fonctions qui trient une liste  
mesurer le temps que mettent ces fonctions  
afficher des graphiques pour comparer les vitesses  

## Les fonctions 

`recherche_lineaire` : cherche un élément en regardant tout.  
 `recherche_dichotomique` : cherche plus vite mais la liste doit être triée.  
`tri_selection` : trie la liste lentement.  
`tri_rapide` : trie la liste très vite.

## Ce que le programme affiche

- un graphique qui compare les deux recherches  
- un graphique qui compare les deux tris
